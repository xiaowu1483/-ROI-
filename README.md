# [重要] 项目说明书
# 📊 电商广告 ROI 与利润自动化分析系统

> 一个基于 Python 的电商数据分析工具，帮助运营人员监控广告投放效果、分析商品利润、识别亏损商品。

---

##  项目背景

在电商运营中，广告投放效果监控和商品利润分析是日常工作的重要部分。本项目旨在通过自动化数据分析流程，帮助运营人员：

- **快速计算** 广告 ROI、转化率、利润率等核心指标
- **可视化展示** 销售趋势和商品表现
- **自动预警** 亏损商品，辅助决策调整

---

## ✨ 核心功能

| 功能模块 | 描述 |
|----------|------|
| 📈 指标计算 | 自动计算 ROI、CPC、CVR、利润率等电商核心指标 |
| 📊 可视化看板 | 交互式数据看板，支持筛选和钻取分析 |
| ⚠️ 亏损预警 | 自动识别平均利润为负的商品并预警 |
| 💾 数据管理 | SQLite 数据库存储，支持历史数据查询 |
| 📥 数据导出 | 支持筛选后数据导出为 CSV |

---

## 🛠️ 技术栈

- **编程语言**: Python 3.8+
- **数据处理**: Pandas, NumPy
- **数据库**: SQLite3
- **可视化**: Plotly, Streamlit
- **项目管理**: Git, GitHub
